On Windows make sure to change Interpreter to miniconda interpreter with Tensorflow < 2.11
Python version has to be <=3.9 depending on your version of Tensorflow

Windows cannot run the latest version of tensorflow and use latest version of CUDA/ cuDNN

Everything has to be downgraded to run natively on windows or you can use a virtual machine
to run the latest version of the software on linux

Tensorflow install instructiona found here: https://www.tensorflow.org/install/pip?hl=en#windows-native_1
